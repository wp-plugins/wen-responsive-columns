== Description ==

<h3>WEN Responsive Columns</h3>WEN Responsive Columns provide you a shortcode which will help you display columnized content. Onle one shortcode `[wrc_column]` is there in the plugin. But one shortcode is enough for generating multiple mixex of columns. Grid could be 2,3,4,6 or 12. You can easily add columns using button in the editor. Select your desired layout and and shortcode. That's it.

Plugin: [Home Page](http://wenthemes.com/item/wordpress-plugins/wen-responsive-columns/)

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for `wen-responsive-columns`
3. Click `Install Now`
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `wen-responsive-columns.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `wen-responsive-columns.zip`
2. Extract the `wen-responsive-columns` directory to your computer
3. Upload the `wen-responsive-columns` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
